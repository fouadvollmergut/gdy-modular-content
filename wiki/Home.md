## About GDY Modular Content

GDY Modular Content is a WordPress plugin that allows you to create modular content and edit it with an intuitive user interface from the frontend of your site. The plugin's main philosophy is to make content management easy for everyone:

- **Easy handling for editors** - Intuitive frontend interface for content management
- **Fast and flexible implementation for programmers** - Clean API and extensible architecture
- **Good control over layouts for designers** - Flexible module system with full design control

The plugin uses the native WordPress architecture to save content (postmeta, attachments), ensuring compatibility with most WordPress environments.

## Features

- Frontend content editing with an intuitive user interface
- Modular content system with reusable components
- Native WordPress architecture for content storage
- Support for various content types (text, images, galleries, tables, etc.)
- Preview modes (soft and hard preview)
- Module visibility and timing controls
- Batch operations for module management
- Child theme support
- Multilingual support with WordPress admin language

## Requirements

- WordPress 3.6 or higher (tested up to 6.7.2)
- PHP 5.6 or higher
- User capability: `edit_pages`
